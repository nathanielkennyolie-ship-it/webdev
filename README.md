# Nathaniel Kenny Olie — Freelance Web Developer Portfolio

Personal portfolio site for Nathaniel Kenny Olie, a freelance web developer based in Warri, Delta State, Nigeria.

## Sections

- **Home** — Hero with personal branding and CTAs
- **About** — Bio, experience, and skills
- **Services** — Hotel, church, business, and website rescue
- **Projects** — Live project showcase with cards and links
- **Pricing** — Transparent pricing tiers (Starter, Standard, Premium)
- **CV** — Resume highlights with download link
- **Contact** — Contact form + WhatsApp integration

## Tech

- Pure HTML + CSS (inline)
- Vanilla JavaScript for interactivity
- Fully responsive (mobile-first breakpoints)
- SEO-optimized meta tags and semantic HTML
- 99/100 PageSpeed target

## Customize

Replace placeholder content:
- Update the WhatsApp number (`+234 800 000 0000`) throughout
- Replace the hero background image (`hero-bg.webp`)
- Update email address and social links
- Add your own project URLs
- Upload your CV PDF and update the download link
